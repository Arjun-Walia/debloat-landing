PROJECT OVERVIEW
--------------------------------------------------------------------------------
Debloat AI is a desktop application (Windows/Mac/Linux) that connects to 
Android devices via ADB. Users remove phone bloatware by chatting with an AI 
powered by Perplexity. This landing page communicates that product vision.

DESIGN SYSTEM: RAW CYBER-BRUTALISM
--------------------------------------------------------------------------------
- ZERO glassmorphism, zero soft shadows, zero blur effects
- Hard 1px borders (#1a1a1a)
- Monospace typography throughout (Spline Sans Mono)
- Color palette: #030303 (bg), #D33C34 (red), #71A1A1 (teal), #f97316 (orange)
- Noise texture overlay for physical/raw aesthetic
- 60px grid background pattern in hero section

TYPOGRAPHY
--------------------------------------------------------------------------------
- Headlines: Spline Sans Mono, 700 weight
- Body: Spline Sans Mono for consistency
- Display sizes: text-5xl to text-7xl for hero headlines
- Labels: 0.65rem, uppercase, 0.2em letter-spacing

TECH STACK
--------------------------------------------------------------------------------
- Next.js 16.1.6 with React 19
- Tailwind CSS v4
- Framer Motion v12.34.3 (0.2s animations, circOut easing)
- Lucide React v0.575.0 for icons
- TypeScript
